This folder contains flag assets referenced by the UI.
Add ISO country code images, e.g.:
- us.svg and us.png
- gb.svg and gb.png
- fr.svg and fr.png
- de.svg and de.png
- it.svg and it.png
- cn.svg and cn.png
- il.svg and il.png

If an SVG is missing, the UI falls back to an inline emoji SVG or displays the emoji in a circle.
